https://jp_funda.com(仮)

JP Funda Api Documentation

このWebAPIでは、日本企業の有価証券報告書の情報を取得することができます。

どのURLにアクセスしても、有価証券報告書をレスポンスとして返しますが、 エンドポイントによって証券コードやEDINETコードなど、取得するデータを指定することができます。

NOTE: 現在はテストとして公開しています。 NOTE: 使用するさいは、APIキーを作成する必要があります。

Headerについて ¶

このAPIは認証トークン(APIキー)が必要です。User登録をして、APIキーを発行して下さい。 各APIには次のようにヘッダにトークンを設定します。 NOTE: 最大100回まで無料でアクセスすることができます。(取得する有価証券報告書の数で制限を設けています。)

Authorization: Token APIキー

証券コードでデータ取得 ¶

証券コードに対応する企業の最新の有価証券報告書のデータを取得できます。 最新の有価証券報告書のみ取得するURLと、全ての年度のデータを取得するURLがあります。

最新の有価証券報告書のみ  ¶

データを取得したい企業の{証券コード}４桁をurlの最後尾に入れてください。

最新の有価証券報告書のデータのみ
GET/api/jp/securities_code/{securities_code}

証券コードに対応する企業の公開されている最新の有価証券報告書のデータを返します。

Example URI
