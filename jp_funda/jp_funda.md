# JP Funda Api Documentation
このWebAPIでは、日本企業の有価証券報告書の情報を取得することができます。

どのURLにアクセスしても、有価証券報告書をレスポンスとして返しますが、
エンドポイントによって証券コードやEDINETコードなど、取得するデータを指定することができます。


NOTE: 現在はテストとして公開しています。
NOTE: 使用するさいは、APIキーを作成する必要があります。


## Headerについて

このAPIは認証トークン(APIキー)が必要です。User登録をして、APIキーを発行して下さい。
各APIには次のようにヘッダにトークンを設定します。
NOTE: 最大100回まで無料でアクセスすることができます。(取得する有価証券報告書の数で制限を設けています。)

```
Authorization: Token APIキー
```
